# Medical Marketing Agency — Website

Landing page for **Medical Marketing Agency**, a Romanian consultancy specialising in medical and pharmaceutical marketing, administrative representation, and clinic/pharmacy licensing.

## Stack

- **[Astro](https://astro.build/) 6** — static site generator, zero JS by default
- Vanilla CSS with a full design-token system (spacing, color, typography)
- Custom variable fonts: Playfair Display (display) + Jost (sans)

## Sections

| Section | Description |
|---|---|
| Hero | Headline + dual CTA (call / how it works) |
| Trust | Compliance badges — GDP/GMP, ANMDMR, Colegiul Medicilor, DSP |
| Services | Six core service cards |
| Client Types | Who the agency works with |
| Benefits | Why choose MMA — local presence, medical focus, client references |
| Process | Four-step engagement flow |
| Q&A | Accordion FAQ + image panel |
| Reviews | Client testimonials |
| Footer | Brand, services list, address, contact |

## Development

```bash
npm install
npm run dev       # localhost:4321
npm run build     # production build → dist/
npm run preview   # preview production build
```

Requires **Node ≥ 22.12**.
